Malibu
A mixture of Pinterest and Imgur for learning English.
It's a pin board filled with entertaining posts translated to Russian.
The killer feature of this project is a highlighter for new words and phrases allowing you to save them and train afterwards:
The same post is shown but previously highlighted part is replaced with an input box so you have to recall the missing part based on the context and visual associations and type it in.
An exciting idea which received a huge amount of positive feedback but unfortunately we haven't found a way to make a business out of it.
  • PHP
  • Javascript
  • jQuery
  • Google APIs
  • Yandex APIs
