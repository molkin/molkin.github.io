DB Explorer
While working on accounting side of things at DeviantArt I had to jump a lot between mysql tables especially when having to investigate what happened to an order/transaction and why we ended in the current state. Such a task would have me start with looking up orders by order group id (what looks like one order to the user might contain multiple orders that go to different manufacturers) then getting order items from a db table that doesn't have self-speaking info but ids linking to auxiliary tables e.g. itemtypeid, merchantid, productid, subproductid.
So I created a tool for quick database lookups in CLI style with autosuggest and simple query syntax which allows to quickly jump from an (id|column of ids) from one table to another table filtered by the selected id(s).
It was also a great opportunity to practice React/Flux.
